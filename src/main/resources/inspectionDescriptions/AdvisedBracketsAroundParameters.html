Reports when a named parameter has multiple values, but they are not contained within brackets.

Example:


data json: "..."
| summarize field1, field2, by: field3, field4 // "by" parameter is the problem

After the quick-fix is applied, value of the by parameter will be contained within brackets:


data json: "..."
| summarize field1, field2, by: { field3, field4 }

It is advised to list fields in bracket expression for a better code quality and readability.
