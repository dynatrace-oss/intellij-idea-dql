Validates if parameter values are chained together.

DQL allows variadic parameter values to be disjointed from each other. This notation creates great confusion and makes it harder to understand the parameter definition. This inspection will report an issue if it finds a parameter with multiple disjointed values groups. 


data record()
| summarize f1, f2, by: f3, f4 // f4 is still treated as an aggregation
After applying the quick fix, parameter values will be chained together. 

data record()
| summarize f1, f2, f4, by: f3
