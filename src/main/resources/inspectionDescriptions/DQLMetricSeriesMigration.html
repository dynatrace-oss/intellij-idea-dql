Verifies if the "metrics" command could be used.

The metric.series data object was marked as deprecated and was replaced with a dedicated command named metrics. All usages should migrate to a new solution.

This inspection reports an issue if it detects usage of fetch metric.series. 


fetch metric.series, from: -4h, to: -1h, bucket: "test", samplingRatio: 5, scanLimitGBytes: 1

After applying the quick-fix, the command will be automatically updated to "metrics". 


metrics from: -4h, to: -1h, bucket: "test"
