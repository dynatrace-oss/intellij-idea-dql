Checks if the negation expression uses a boolean value.

To be a deemed valid expression, the negation parameter needs to be used on a boolean value, or on an expression that returns the boolean value. See all supported operation types in the official Dynatrace documentation. 


data json: "..."
| filter not true // this is valid, uses a boolean value
| filter not 55 // this is invalid, provided a positive long value that cannot be negated
| filter somefield // this is valid, if the field contains a boolean value
| filter isNotNull(somefield) // this is valid, because `isNotNull` returns a boolean value
