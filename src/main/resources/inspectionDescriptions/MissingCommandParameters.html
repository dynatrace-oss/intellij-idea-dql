Verifies if a command has a properly defined parameters list.

Will report an error if any of the required parameters is missing.


data json: "..."
| lookup [...] // is missing parameters "sourceField" and "lookupField"

After applying the quick-fix, all required parameters will be added and the user will be prompted to fill in their values. 


data json: "..."
| lookup [...], sourceField: some.field, lookupField: some.field // the "some.field" will be template variables for the user to fill in
