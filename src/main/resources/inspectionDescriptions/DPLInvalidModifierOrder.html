Verifies if the expression definition contains modifiers specified in a correct order.

This inspection reports an error if it detects an expression that specifies modifiers in a wrong place.

The correct order of modifiers:

  1. Lookaround
  2. Base expression (command/group/character class etc.)
  3. Matchers ({...})
  4. Configuration (...)
  5. Quantifiers (*, +, or {...})
  6. Optional symbol (?)

// The order of optionality, quantifier and configuration modifiers is wrong
INT?*(min=0)
After applying the quick fix, the expression will reordered.
