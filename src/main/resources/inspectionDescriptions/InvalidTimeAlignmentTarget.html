Validates the duration type used in the time alignment operation.

This inspection reports an error when an unknown duration type is used for the expression.

DQL offers a list of possible duration shortcuts. You can find the list of supported duration types in the official Dynatrace documentation. 


fetch logs,
    from: now@d, // "d" is a valid time alignment operand
    to: now@nextWeek // "nextWeek" is not a valid time alignment operand
