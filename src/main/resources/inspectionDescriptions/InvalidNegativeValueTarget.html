Checks if the element can be a negative value.

The inspection reports an error for every element that is a negative value, but it cannot be one. Only dynamic fields, functions, numbers and durations can be preceded with - sign. 


data json: "..."
| fields {
    f1 = -5,
    f2 = -5d,
    f3 = -max(5,1),
    f4 = -"string" // will raise an error, because the string cannot be a negative value
}
