Verifies if the configuration was defined only once.

This inspection reports an error if it detects a parameter that was already set up. 


'// This reports an error, because the parameter "tz" is an alias of "timezone", so it is duplicated
TIMESTAMP("yyyy-MM-dd HH:mm:ss.S", tz="UTC", timezone="UTC")
After applying the quick fix, the duplicated option will be dropped.
