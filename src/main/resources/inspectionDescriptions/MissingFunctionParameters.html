Verifies if a function has all required parameters set.

Will report an error if any of the required parameters is missing. 


fetch logs
    | filter matchesValue(field) // will report an error, because the "value" parameter is missing

After applying the quick-fix all required parameters will be added and the user will be prompted to fill in their values. 


fetch logs
    | filter matchesValue(field, value) // the "value" parameter will be a template variable for the user to fill in
