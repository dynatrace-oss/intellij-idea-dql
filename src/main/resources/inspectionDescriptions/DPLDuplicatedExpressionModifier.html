Verifies if the expression modifier was applied only once.

This inspection reports an error if it detects an expression that already specified the type of modifier. 


// The INT expression provided two types of quantifiers: simple ("*") and limited ("{1,2})
INT{1,2}*
// The ENUM expression provided two matchers instead of just one
ENUM{'success'=0}{'error'=1}
// The INT expression provided two configuration blocks
INT(min=0)(max=1)
After applying the quick fix, the duplicated modifier will be dropped.
