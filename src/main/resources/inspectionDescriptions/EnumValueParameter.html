Ensures a correct enumeration value is used.

Some parameters have a predefined set of values that they can be assigned to. This inspection reports an error when the parameter value is not on the enumeration values list. 


data "..."
| lookup [
    data "..."
],
  sourceField: some.field,
  lookupField: some.field,
  executionOrder: invalidValue // execution order has a predefined set of values, so "invalidValue" will report an error.
