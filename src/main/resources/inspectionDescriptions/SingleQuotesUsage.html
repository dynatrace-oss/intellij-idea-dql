Verifies that no single-quoted strings are used in the DQL query.

DQL does not allow single-quoted strings usage. This inspection will report an error if it finds any such string used anywhere in the query. 


data json: "..."
    | fields field = 'an invalid string' // single quotes are not supported

After applying the quick-fix, single quotes will be automatically replaced with double ones, using a correct type of DQL string. 


data json: "..."
    | fields field = "an invalid string" // single quotes will be automatically converted into the double quotes
